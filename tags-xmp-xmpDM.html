XMP Dynamic Media Schema

This schema specifies XMP properties used by the Adobe dynamic media group.

  • Exiv2 keys are Xmp.xmpDM.<Property>
  • The schema namespace URI is http://ns.adobe.com/xmp/1.0/DynamicMedia/
  • The preferred schema namespace prefix is xmpDM
Reference: XMP Specification and Documentation

Property Label Value type Exiv2 type Category Description
absPeakAudioFilePath Absolute Peak Audio File Path URI XmpText Internal The absolute path to the file's peak audio file. If empty, no peak file exists.
album Album Text XmpText External The name of the album.
altTapeName Alternative Tape Name Text XmpText External An alternative tape name, set via the project window or timecode dialog in Premiere. If an alternative name has been set and has not been reverted, that name is displayed.
altTimecode Alternative Time code Timecode XmpText External A timecode set by the user. When specified, it is used instead of the startTimecode.
artist Artist Text XmpText External The name of the artist or artists.
audioModDate Audio Modified Date Date XmpText Internal (deprecated) The date and time when the audio was last modified.
audioChannelType Audio Channel Type closed Choice of Text XmpText Internal The audio channel type. One of: Mono, Stereo, 5.1, 7.1, 16 Channel, Other.
audioCompressor Audio Compressor Text XmpText Internal The audio compression used. For example, MP3.
audioSampleRate Audio Sample Rate Integer XmpText Internal The audio sample rate. Can be any value, but commonly 32000, 44100, or 48000.
audioSampleType Audio Sample Type closed Choice of Text XmpText Internal The audio sample type. One of: 8Int, 16Int, 24Int, 32Int, 32Float, Compressed, Packed, Other.
beatSpliceParams Beat Splice Parameters beatSpliceStretch XmpText Internal Additional parameters for Beat Splice stretch mode.
cameraAngle Camera Angle open Choice of Text XmpText External The orientation of the camera to the subject in a static shot, from a fixed set of industry standard terminology. Predefined values include:Low Angle, Eye Level, High Angle, Overhead Shot, Birds Eye Shot, Dutch Angle, POV, Over the Shoulder, Reaction Shot.
cameraLabel Camera Label Text XmpText External A description of the camera used for a shoot. Can be any string, but is usually simply a number, for example "1", "2", or more explicitly "Camera 1".
cameraModel Camera Model Text XmpText External The make and model of the camera used for a shoot.
cameraMove Camera Move open Choice of Text XmpText External The movement of the camera during the shot, from a fixed set of industry standard terminology. Predefined values include: Aerial, Boom Up, Boom Down, Crane Up, Crane Down, Dolly In, Dolly Out, Pan Left, Pan Right, Pedestal Up, Pedestal Down, Tilt Up, Tilt Down, Tracking, Truck Left, Truck Right, Zoom In, Zoom Out.
client Client Text XmpText External The client for the job of which this shot or take is a part.
comment Comment Text XmpText External A user's comments.
composer Composer Text XmpText External The composer's name.
contributedMedia Contributed Media bag Media XmpBag Internal An unordered list of all media used to create this media.
copyright Copyright Text XmpText External (Deprecated in favour of dc:rights.) The copyright information.
director Director Text XmpText External The director of the scene.
directorPhotography Director Photography Text XmpText External The director of photography for the scene.
duration Duration Time XmpText Internal The duration of the media file.
engineer Engineer Text XmpText External The engineer's name.
fileDataRate File Data Rate Rational XmpText Internal The file data rate in megabytes per second. For example: "36/10" = 3.6 MB/sec
genre Genre Text XmpText External The name of the genre.
good Good Boolean XmpText External A checkbox for tracking whether a shot is a keeper.
instrument Instrument Text XmpText External The musical instrument.
introTime Intro Time Time XmpText Internal The duration of lead time for queuing music.
key Key closed Choice of Text XmpText Internal The audio's musical key. One of: C, C#, D, D#, E, F, F#, G, G#, A, A#, B.
logComment Log Comment Text XmpText External User's log comments.
loop Loop Boolean XmpText Internal When true, the clip can be looped seamlessly.
numberOfBeats Number Of Beats Real XmpText Internal The number of beats.
markers Markers seq Marker XmpSeq Internal An ordered list of markers
metadataModDate Metadata Modified Date Date XmpText Internal (deprecated) The date and time when the metadata was last modified.
outCue Out Cue Time XmpText Internal The time at which to fade out.
projectName Project Name Text XmpText External The name of the project of which this file is a part.
projectRef Project Reference ProjectLink XmpText Internal A reference to the project that created this file.
pullDown Pull Down closed Choice of Text XmpText Internal The sampling phase of film to be converted to video (pull-down). One of: WSSWW, SSWWW, SWWWS, WWWSS, WWSSW, WSSWW_24p, SSWWW_24p, SWWWS_24p, WWWSS_24p, WWSSW_24p.
relativePeakAudioFilePath Relative Peak Audio File Path URI XmpText Internal The relative path to the file's peak audio file. If empty, no peak file exists.
relativeTimestamp Relative Timestamp Time XmpText Internal The start time of the media inside the audio project.
releaseDate Release Date Date XmpText External The date the title was released.
resampleParams Resample Parameters resampleStretch XmpText Internal Additional parameters for Resample stretch mode.
scaleType Scale Type closed Choice of Text XmpText Internal The musical scale used in the music. One of: Major, Minor, Both, Neither.
scene Scene Text XmpText External The name of the scene.
shotDate Shot Date Date XmpText External The date and time when the video was shot.
shotDay Shot Day Text XmpText External The day in a multiday shoot. For example: "Day 2", "Friday".
shotLocation Shot Location Text XmpText External The name of the location where the video was shot. For example: "Oktoberfest, Munich Germany" For more accurate positioning, use the EXIF GPS values.
shotName Shot Name Text XmpText External The name of the shot or take.
shotNumber Shot Number Text XmpText External The position of the shot in a script or production, relative to other shots. For example: 1, 2, 1a, 1b, 1.1, 1.2.
shotSize Shot Size open Choice of Text XmpText External The size or scale of the shot framing, from a fixed set of industry standard terminology. Predefined values include: ECU --extreme close-up, MCU -- medium close-up. CU -- close-up, MS -- medium shot, WS -- wide shot, MWS -- medium wide shot, EWS -- extreme wide shot.
speakerPlacement Speaker Placement Text XmpText External A description of the speaker angles from center front in degrees. For example: "Left = -30, Right = 30, Center = 0, LFE = 45, Left Surround = -110, Right Surround = 110"
startTimecode Start Time Code Timecode XmpText Internal The timecode of the first frame of video in the file, as obtained from the device control.
stretchMode Stretch Mode closed Choice of Text XmpText Internal The audio stretch mode. One of: Fixed length, Time-Scale, Resample, Beat Splice, Hybrid.
takeNumber Take Number Integer XmpText External A numeric value indicating the absolute number of a take.
tapeName Tape Name Text XmpText External The name of the tape from which the clip was captured, as set during the capture process.
tempo Tempo Real XmpText Internal The audio's tempo.
timeScaleParams Time Scale Parameters timeScaleStretch XmpText Internal Additional parameters for Time-Scale stretch mode.
timeSignature Time Signature closed Choice of Text XmpText Internal The time signature of the music. One of: 2/4, 3/4, 4/4, 5/4, 7/4, 6/8, 9/8, 12/8, other.
trackNumber Track Number Integer XmpText External A numeric value indicating the order of the audio file within its original recording.
tracks Tracks bag Track XmpBag Internal An unordered list of tracks. A track is a named set of markers, which can specify a frame rate for all markers in the set. See also xmpDM:markers.
videoAlphaMode Video Alpha Mode closed Choice of Text XmpText External The alpha mode. One of: straight, pre-multiplied.
videoAlphaPremultipleColor Video Alpha Premultiple Color Colorant XmpText External A color in CMYK or RGB to be used as the pre-multiple color when alpha mode is pre-multiplied.
videoAlphaUnityIsTransparent Video Alpha Unity Is Transparent Boolean XmpText Internal When true, unity is clear, when false, it is opaque.
videoColorSpace Video Color Space closed Choice of Text XmpText Internal The color space. One of: sRGB (used by Photoshop), CCIR-601 (used for NTSC), CCIR-709 (used for HD).
videoCompressor Video Compressor Text XmpText Internal Video compression used. For example, jpeg.
videoFieldOrder Video Field Order closed Choice of Text XmpText Internal The field order for video. One of: Upper, Lower, Progressive.
videoFrameRate Video Frame Rate open Choice of Text XmpText Internal The video frame rate. One of: 24, NTSC, PAL.
videoFrameSize Video Frame Size Dimensions XmpText Internal The frame size. For example: w:720, h: 480, unit:pixels
videoModDate Video Modified Date Date XmpText Internal (deprecated)The date and time when the video was last modified.
videoPixelDepth Video Pixel Depth closed Choice of Text XmpText Internal The size in bits of each color component of a pixel. Standard Windows 32-bit pixels have 8 bits per component. One of: 8Int, 16Int, 32Int, 32Float.
videoPixelAspectRatio Video Pixel Aspect Ratio Rational XmpText Internal The aspect ratio, expressed as ht/wd. For example: "648/720" = 0.9
partOfCompilation Part Of Compilation Boolean XmpText External Part of compilation.
lyrics Lyrics Text XmpText External Lyrics text. No association with timecode.
discNumber Disc Number Text XmpText External If in a multi-disc set, might contain total number of discs. For example: 2/3.
